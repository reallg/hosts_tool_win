## 关于

hosts_tool_win 是一个自动从网络下载，替换windows系统hosts文件的脚本


## 使用

- 编辑脚本，设置自己需要的hosts源地址

```bat
set HostsSource="http://your_raw_hosts_file"
```

- hosts失效的双击运行一下
- 搞定！
